

## Commit emoji convention
#
# | emoji              | description                             |
# |--------------------|-----------------------------------------|
# | :memo:             | Add comment or doc                      |
# | :gift:             | New feature.                            |
# | :bug:              | Bug fix                                 |
# | :bomb:             | Breaking compatibility.                 |
# | :white_check_mark: | Write test.                             |
# | :fire:             | Remove something.                       |
# | :beer:             | I'm happy like reduced code complexity. |
